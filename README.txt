Easy Woocommerce MailChimp Integration with Ecommerce 360 and Merge Fields.

== Description ==

= INTEGRATE MAILCHIMP WITH WOOCOMMERCE WITHOUT ANY HASSLE =


WooCommerce Newsletter Subscription is a powerful plugin that provides MailChimp integration with Ecommerce 360 and Merge Fields. It uses latest MailChimp API to automatically subscribe customers and has Ecommerce 360 compatibility to track sales back into MailChimp.

You can use it as a simple e-mail subscription plugin or capture data of purchase for up selling and targeted customer automation mailers from MailChimp.


= WHY YOU SHOULD BE USING WOOCOMMERCE NEWSLETTER SUBSCRIPTION =


= OPT-IN SETTINGS  =

With the amazing opt-in settings of this plugin, you can choose whether you want to subscribe the customer after their purchase automatically. You can choose either single or double opt-in (customer has to confirm before subscribing). This plugin will enable you to make custom labels for subscription form which will be displayed on checkout.

= ECOMMERCE 360 =

WooCommerce Newsletter Subscription plugin comes with Ecommerce 360 enabled. The customer purchase will be automatically sent to MailChimp.

= MERGE TAGS =

Merge tags enable you to add fields such as Role, Country, Purchase Value, etc. to your MailChimp list. The fields will be automatically fetched. It has easy mapping features.



= Requirements =

WooCommerce Newsletter Subscription recommends latest PHP 7.0+ and WordPress Versions. Minimum requirements - PHP 5.4+, WordPress 3.5.1+ and WooCommerce 2.2+.

== Installation ==

1. Upload or extract the `woocommerce-newsletter-subscription` folder to your site's `/wp-content/plugins/` directory. You can also use the *Add new- option found in the *Plugins- menu in WordPress.
2. Enable the plugin from the *Plugins- menu in WordPress.

= Usage =

1. Go to WooCommerce > Newsletter Subscription
2. First, paste your MailChimp API Key and hit `Save changes` to get started.
3. Next, select your MailChimp list.
4. Select your opt-in, double opt-in settings and hit `Save changes`.
5. That's it, your WooCommerce store is now integrated with MailChimp!
